Insert an item into the tree.

This function inserts the designated item into the proper position of the tree to maintain the structure correctness.

Parameters
selfThe pointer to the SimpleTree structure

Return values
SUCC
ERR_NOMEMInsufficient memory for item insertion
Note
If the order of the designated item is the same with a certain one stored in the tree, the old item will be cleaned by the resource deallocation function so that the designated one can take the position.
